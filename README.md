# Interface-Arduino-Visual-Studio-2013
Proyek kali ini saya dedikasikan untuk tugas akhir (SKRIPSI) dan sebagai ilmu pembelajaran untuk kalian yang sedang membutuhkan cara membuat interface pengukuran arduino ke Visual Studio 2013 dan database menggunakan Ms. Access 2010/2007.

Silahkan download semua file yang saya sediakan sebagai bahan belajar dan ilmu yang bermanfaat untuk kalian.

Apabila teman-teman ingin membuat alat serupa dapat mencetak design PCB buatan saya silahkan kunjungi link berikut:
https://www.pcbway.com/project/shareproject/Datalogger_pH_and_Temperature.html

Pada link tersebut saya menyedikan file schematic dan board rangkaian secara gratis. Alat dan bahan yang saya gunakan telah saya cantumkan pada link di atas.
